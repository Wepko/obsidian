
### Анализ BIOS (UEFI)

В ходе работы были проанализированы экраны BIOS UEFI материнской платы MSI MPG B550 GAMING PLUS. Рассмотрим значение ключевых пунктов на нескольких выбранных изображениях.

#### Экран 1: Главный экран (Status / Info)

**Рисунок 1 – Главный экран BIOS (Status)**

Данный экран (фото `photo_19_2026...`) является стартовым (часто называемым EZ Mode или Status). Он отображает текущее состояние системы и основные параметры мониторинга.

*   **CLICK BIOS S**: Название графической оболочки UEFI от компании MSI. Это аналог MyASUS in UEFI из примера, только для материнских плат MSI.
*   **Информация о системе (верхняя панель):**
    *   **CPU Speed (4.60 GHz)**: Текущая частота процессора. Значение 4.60 GHz указывает на то, что процессор AMD Ryzen 7 5700G работает в режиме штатного авторазгона (Boost).
    *   **DDR Speed (3200 MHz)**: Текущая частота оперативной памяти. Значение 3200 MHz говорит о том, что активирован профиль XMP (или эквивалент для AMD - A-XMP), так как базовые частоты для DDR4 обычно ниже.
*   **Мониторинг температур и напряжений (центральный блок справа):**
    *   **CPU Core Temperature (55°C)**: Температура ядер процессора в данный момент. Находится в пределах нормы.
    *   **Motherboard Temperature (31°C)**: Температура чипсета или зоны VRM (системы питания) материнской платы.
    *   **VCore (1.430V)**: Напряжение, подаваемое на ядра процессора. Значение повышено (выше стандартного 1.2-1.3V), что характерно для режима автоматического разгона или завышенных настроек BIOS (Game Boost).
*   **Информация об оборудовании (нижний блок):**
    *   **MB: MPG B550 GAMING PLUS (MS-7C56)**: Точная модель материнской платы.
    *   **CPU: AMD Ryzen 7 5700G with Radeon Graphics**: Модель процессора со встроенной графикой (APU).
    *   **BIOS Ver / Build Date**: Версия прошивки BIOS E7C56AMS.1A0 от 21 мая 2022 года. Это указывает на то, что прошивка не самая свежая и требует обновления для поддержки новых процессоров или исправления уязвимостей.

#### Экран 2: Вкладка «Напряжения» (Voltage Status)

**Рисунок 2 – Мониторинг напряжений (Voltage)**

На всех скриншотах справа присутствует блок **Voltage**. Он критически важен для оценки стабильности работы ПК.

*   **CPU Core (1.430V)**: Как уже было сказано, довольно высокое напряжение на ядре. При длительной высокой нагрузке это может привести к перегреву, если используется слабое охлаждение.
*   **CPU NB/SOC (1.826V)**: **КРИТИЧЕСКИ ВАЖНЫЙ ПУНКТ.** NB/SOC (Northbridge / System on Chip) — это напряжение контроллера памяти и инфраструктуры внутри процессора.
    *   *Анализ:* Значение **1.826V** является аномально высоким для процессоров AMD Ryzen (безопасный предел обычно до 1.2-1.25V). Это может быть причиной нестабильности, перегрева или даже выхода из строя процессора или материнской платы. Скорее всего, это ошибка отображения датчика (особенность прошивки) либо серьезная ошибка в настройках (режим LN2 для экстремального разгона). Пользователю следует немедленно проверить эти настройки.
*   **DRAM (1.340V)**: Напряжение питания оперативной памяти. Для DDR4 значение 1.35V является стандартным для большинства профилей XMP (разгона), что соответствует частоте 3200 MHz.
*   **Системные напряжения (3.3V, 5V, 12V)**: Показывают, что блок питания выдает стабильные напряжения в допустимых пределах (например, 12.192V при норме 12V — это отлично).

#### Экран 3: Вкладка «Дополнительно / Периферия» (Advanced / Integrated Peripherals)

**Рисунок 3 – Экран вкладки «Дополнительно / Встроенные периферийные устройства»**

Этот экран (фото `photo_8_2026...`) находится на вкладке «Дополнительно» (Advanced) и отвечает за управление встроенными контроллерами.

*   **Конфигурация встроенного ЛВС:**
    *   **Встроенный контроллер LAN [Включено]**: Сетевая карта (Ethernet) работает. Если выключить, интернет по проводу пропадет.
    *   **ПЗУ LAN [Выключено]**: Опция загрузки компьютера по сети (PXE). Отключена, что ускоряет загрузку системы, если не используется бездисковая рабочая станция.
*   **Конфигурация SATA:**
    *   **SATA режим [AHCI режим]**: Правильный режим для современных SSD и HDD. Режим RAID не активирован, так как диски работают по отдельности.
    *   **SATAx Hot Plug [Выключено]**: Поддержка «горячей» замены дисков отключена. Включение требуется только для серверов или внешних корзин с дисками.
*   **Конфигурация Аудио:**
    *   **HD Аудиоконтроллер [Включено]**: Встроенная звуковая карта активна. Если используется дискретная звуковая карта в PCIe слоте, здесь её можно отключить для экономии ресурсов.

#### Экран 4: Вкладка «Загрузка» (Boot)

**Рисунок 4 – Экран вкладки «Загрузка» (Boot Configuration)**

На этом скриншоте (фото `photo_16_2026...`) представлена вкладка Загрузка, где настраивается очередность запуска ОС.

*   **Boot mode select [UEFI]**: Система загружается в режиме UEFI. Это современный стандарт, поддерживающий GPT-диски и Secure Boot. Второй режим (CSM/Legacy) отключен, хотя в статусе системы указан "CSM/UEFI", что означает совместимость с обоими типами дисков.
*   **FIXED BOOT ORDER Priorities (Приоритет загрузки):**
    *   **Boot Option #1 [UEFI Hard Disk:u...]**: Первым в очереди стоит жесткий диск (SSD) с установленным Windows Boot Manager. Это стандартная настройка.
    *   **Boot Option #2 [UEFI CD/DVD]**: Далее идет привод оптических дисков (если он есть).
    *   **Boot Option #3-#6 [UEFI USB ...]**: Различные типы USB-носителей. Именно эти пункты позволяют загрузиться с флешки для переустановки системы, не отключая "Fast Boot".

#### Экран 5: Вкладка «Безопасность» (Security)

**Рисунок 5 – Экран вкладки «Безопасность» (Security)**

Этот экран (фото `photo_18_2026...`) отвечает за безопасность системы и установку паролей.

*   **Пароль Администратора / Пользовательский пароль:**
    *   Статус **Не установлено**. Это стандартное состояние для домашнего ПК. Пароль администратора защищает настройки BIOS от изменений, а пользовательский требует ввода при каждой загрузке ПК для доступа к ОС.
*   **Trusted Computing**: Раздел настроек модуля TPM (Trusted Platform Module). TPM необходим для работы BitLocker (шифрование диска) и является обязательным требованием для установки Windows 11. Факт наличия этого раздела подтверждает совместимость системы с современными стандартами безопасности.
*   **Вторжение в корпус (Chassis Intrusion)**: Опция, позволяющая обнаружить факт открытия крышки системного блока. При включении, если датчик (кнопка) на корпусе сработает, при следующем включении BIOS выдаст предупреждение.
