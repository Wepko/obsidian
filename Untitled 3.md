### Исследование экранов BIOS

В ходе выполнения практического задания были проанализированы скриншоты BIOS (UEFI) материнской платы MSI MPG B550 GAMING PLUS. Ниже представлено описание значений пунктов меню и информации для каждого уникального изображения.

#### 1. Анализ скриншота `photo_7_2026-02-20_09-52-54.jpg` (Главный экран / Hardware Monitor)

На данном скриншоте представлен главный экран BIOS в режиме отображения информации о системе и датчиках.

**Описание пунктов:**

*   **CPU Speed / DDR Speed:** Отображает текущую частоту работы процессора (4.60 ГГц) и оперативной памяти (3200 МГц).
*   **CPU Core Temperature:** Температура ядер процессора в реальном времени. На момент снимка составляла **51°C**. Это нормальная рабочая температура для системы под небольшой нагрузкой или в простое.
*   **Motherboard Temperature:** Температура чипсета или системы охлаждения материнской платы — **31°C**.
*   **VCore:** Напряжение питания ядра процессора. Здесь указано **1.428V - 1.430V**. Это штатное напряжение для данного процессора (Ryzen 7) в режиме авторазгона или под нагрузкой.
*   **DDR Voltage:** Напряжение питания модулей оперативной памяти — **1.340V**. Соответствует стандарту для DDR4 (обычно 1.2-1.35V).
*   **BIOS Mode:** Режим загрузки BIOS. Указан **CSM/UEFI** (оба режима включены). Это означает, что система поддерживает загрузку как с новых дисков с таблицей GPT (UEFI), так и со старых с MBR (CSM).
*   **MB, CPU, Memory Size, BIOS Ver/Date:** Краткая сводка по установленному оборудованию и версии прошивки BIOS.
*   **Boot Priority / A-XMP Profile:** Значки быстрого доступа. Показывают, что активен (или доступен для выбора) профиль A-XMP Profile 1.
*   **Settings\Дополнительно\Настройки АСРI:**
    *   **Индикатор питания [Мигание]:** Определяет поведение светодиода на корпусе ПК в режиме сна (S3, Suspend to RAM). Режим «Мигание» является стандартным.
    *   **S3/Modern Standby Support [S3 Enable]:** Включает поддержку традиционного режима сна «Suspend to RAM» (S3), при котором питание отключается от большинства компонентов, кроме оперативной памяти.
*   **Voltage (Блок напряжений):**
    *   **CPU NB/SOC (1.824V):** Напряжение северного моста (контроллера памяти), встроенного в процессор.
    *   **CPU VDDP (0.897V):** Напряжение для входных буферов контроллера памяти.
    *   **System 3.3V, 5V, 12V:** Контрольные напряжения блоков питания. Показывают стабильную работу БП (3.396V, 5.090V, 12.192V — в пределах допуска).

#### 2. Анализ скриншота `photo_8_2026-02-20_09-52-54.jpg` (Встроенные периферийные устройства)

Этот экран отвечает за управление контроллерами, впаянными в материнскую плату.

**Описание пунктов:**

*   **VGA Detection [Auto]:** Определяет, нужно ли BIOS'у выводить информацию о видеокарте при запуске.
*   **Конфигурация встроенного ЛВС (LAN):**
    *   **Встроенный контроллер LAN [Включено]:** Включает или отключает сетевую карту (Ethernet), встроенную в материнскую плату.
    *   **ПЗУ LAN [Выключено]:** Отключает возможность загрузки компьютера по сети (PXE/Network Boot). Это повышает безопасность и ускоряет загрузку, если не используется бездисковая станция.
    *   **Network stack [Выключено]:** Отключает стек протоколов UEFI для сети, что также ускоряет POST (самотестирование при включении).
*   **Конфигурация SATA:**
    *   **SATA режим [AHCI режим]:** Режим работы контроллера жестких дисков. AHCI — современный режим, необходимый для работы SSD и корректного функционирования современных ОС (в отличие от устаревшего IDE).
    *   **Hot Plug [Выключено]:** Функция «горячей замены» для каждого из портов SATA (1-6). Отключена для внутренних дисков, чтобы случайно не извлечь их.
*   **Конфигурация Аудио:**
    *   **HD Аудиоконтроллер [Включено]:** Включает встроенную звуковую карту.

#### 3. Анализ скриншота `photo_16_2026-02-20_09-52-54.jpg` (Настройки загрузки)

Данный скриншот показывает меню конфигурации процесса загрузки компьютера.

**Описание пунктов:**

*   **Boot Configuration (Конфигурация загрузки):**
    *   **Отображение логотипа во весь экран:** Включает/выключает заставку (логотип MSI) во время загрузки.
    *   **Bootup NumLock State [Включено]:** Включает индикатор Num Lock на клавиатуре при старте системы, активируя боковую цифровую клавиатуру.
    *   **Info Block effect [On]:** Визуальный эффект в BIOS.
    *   **POST Beep [Выключено]:** Отключает звуковой сигнал динамика при успешном прохождении самотестирования (POST).
*   **Boot mode select [UEFI]:** Выбор режима загрузки. Установлен режим **UEFI**. Это означает, что система будет загружаться только с дисков, размеченных в стиле GPT. (Если бы стояло Legacy, то искала бы загрузчик в стиле MBR).
*   **FIXED BOOT ORDER Priorities (Приоритет загрузки):**
    *   **Boot Option #1 [UEFI Hard Disk]:** Первым в очереди стоит загрузка с жесткого диска (SSD).
    *   **Boot Option #2 [UEFI CD/DVD]:** Вторым — с оптического привода (если есть).
    *   **Boot Option #3-6 [UEFI USB...]:** Далее по очереди различные типы USB-носителей (внешний диск, CD/DVD, Flash-накопитель, Floppy-дисковод).
    *   **Boot Option #7 [UEFI Network]:** Загрузка по сети (последний приоритет).
*   **UEFI Hard Disk Drive BBS Priorities:** Дополнительное меню, позволяющее уточнить, какой именно жесткий диск из списка имеющихся должен быть первым в очереди.

#### 4. Анализ скриншота `photo_15_2026-02-20_09-52-54.jpg` (События пробуждения)

На данном скриншоте представлено меню настроек управления питанием и событий, выводящих систему из спящего режима.

- **Пробуждение по событию [BIOS]:** Указывает, что система может быть разбужена событиями, отслеживаемыми на уровне BIOS.
- **Resume By RTC Alarm [Выключено]:** Функция «будильника». Если включить, компьютер будет автоматически включаться в заданное время (например, каждый день в 9:00). Отключено.
- **Продолжить по команде с PCI устройства [Выключено]:** Запрещает сетевым картам (LAN), контроллерам или другим платам расширения (PCIe) выводить компьютер из сна.
- **Возобновление из S3/S4/S5 с помощью мыши/клавиатуры [Выключено]:** Отключает возможность включения ПК движением мыши или нажатием клавиши на клавиатуре. (Это частая причина случайного включения ПК, поэтому параметр отключен для экономии энергии или предотвращения помех).

#### 5. Анализ скриншота `photo_17_2026-02-20_09-52-54.jpg` (Приоритеты загрузки BBS)

Скриншот показывает подменю выбора конкретного диска для загрузки (UEFI Hard Disk Drive BBS Priorities).

- **Boot Option #1 [bubutu (SATA2: X...)]:** Здесь опечатка в имени (видимо, имелся в виду дистрибутив Linux "Ubuntu"). Это означает, что в качестве первого диска выбран SSD, подключенный к порту SATA2, на котором установлена ОС.
- **Boot Option #2 [Windows Boot Man...]:** Вторым по приоритету стоит загрузчик Windows (скорее всего, находящийся на другом диске или в другом разделе).
- **Boot Option #3 [bubutu (Netac NV...)]:** Третий вариант загрузки — другой накопитель (вероятно, NVMe SSD от производителя Netac), на котором также есть копия Ubuntu.

#### 6. Анализ скриншота `photo_18_2026-02-20_09-52-54.jpg` (Безопасность)

Данный экран отвечает за настройки безопасности и пароли.

- **Пароль Администратора:** Пароль для входа в BIOS и изменения настроек. (Не установлено).
- **Пользовательский пароль:** Пароль для входа в BIOS только с правом просмотра, но без права изменения. (Не установлено).
- **Udisk-Key:** Функция использования USB-флешки в качестве ключа для загрузки системы (своего рода двухфакторная аутентификация на уровне BIOS). (Не установлено).
- **Trusted Computing:** Настройки модуля TPM (Trusted Platform Module), который используется для шифрования дисков (BitLocker) и безопасной загрузки Windows 11.
- **Вторжение в корпус [Выключено]:** Датчик вскрытия крышки системного блока (если он подключен к материнской плате). При включении будет записывать факт вскрытия корпуса.

#### 7. Анализ скриншота `photo_19_2026-02-20_09-52-54.jpg` (Разгон - Overclocking)

Это меню разгона системы (OC Explore Mode).

- **OC Explore Mode [Normal]:** Режим отображения настроек разгона. "Normal" показывает основные параметры, "Expert" — все доступные для тонкой настройки.
- **CPU Ratio / Adjusted CPU Frequency:** Параметры множителя процессора и итоговая частота (4.60 ГГц).
- **DRAM Setting (Profile1/2):** Профили памяти XMP. Указывают тайминги: 16-18-18-38 (CAS Latency - tRCD - tRP - tRAS) и частоту 3200 МГц.
- **Memory Try It!:** Фирменная функция MSI. Позволяет подобрать "быстрые" тайминги для памяти, даже если профиль XMP не загружается стабильно.
- **Memory Fast Boot:** Пропускает полную переинициализацию памяти при каждой загрузке для ускорения старта ПК.

#### 8. Анализ скриншота `photo_20_2026-02-20_09-52-54.jpg` (GAME BOOST)

Здесь показана функция автоматического разгона.

- **GAME BOOST:** Фирменная технология MSI. При активации (переключении ползунка) BIOS автоматически повышает частоты процессора для увеличения производительности в играх. На скриншоте показаны различные предустановленные профили (от 1 до 6).
- **Use USB to flash BIOS / M-FLASH:** Утилита для обновления прошивки BIOS прямо из меню, без загрузки ОС. Позволяет выбрать файл прошивки на флешке.

#### 9. Анализ скриншота `photo_21_2026-02-20_09-52-54.jpg` (Hardware Monitor - Smart Fan)

Это меню управления скоростью вращения вентиляторов (Hardware Monitor).

- **CPU, System, MOS, CHIPSET, PSU:** Температуры различных узлов (Процессор, Корпус, Зона питания процессора, Чипсет, Блок питания). Показаны в градусах Цельсия и Фаренгейта (например 52°C / 125°F).
- **Smart Fan Mode:** Режим работы вентилятора процессора.
    - **PWM:** Режим широтно-импульсной модуляции (4-пиновые вентиляторы) — самое точное управление оборотами.
    - **DC:** Режим управления напряжением (3-пиновые вентиляторы).
    - **Auto:** Автоматическое определение типа вентилятора.
- **График (Chart):** Позволяет вручную настроить кривую зависимости оборотов вентилятора от температуры (точки на графике).

#### 10. Анализ скриншота `photo_11_2026-02-20_09-52-54.jpg` (Конфигурация USB)
Меню управления USB-контроллерами.
- **Устройства USB:** BIOS обнаружил подключенные устройства: "3 Keyboards, 2 Mice, 1 Hub" (3 клавиатуры, 2 мыши, 1 концентратор). Это может быть одна мышь/клавиатура, определяемая как несколько HID-устройств.
- **XHCI Hand-off [Включено]:** Параметр для совместимости со старыми ОС. Передает управление USB 3.0 от BIOS к драйверу ОС. Для Windows 10/11 должен быть включен.
- **Поддержка устаревших USB устройств [Включено]:** Обеспечивает работу клавиатуры и мыши в среде DOS или при установке старых версий Windows. Также позволяет загружаться с USB-флешек в режиме Legacy.

#### 11. Анализ скриншота `photo_12_2026-02-20_09-52-54.jpg` (Super IO Configuration)

Редко используемое меню, отвечающее за устаревшие порты ввода-вывода (COM-порты).
- **Конфигурация последовательного порта 0 (Serial Port):** Настройка COM-порта (RS-232). В современных домашних ПК эти разъемы часто отсутствуют на задней панели, но контакты на материнской плате могут быть. Используется в серверном или промышленном оборудовании.

#### 12. Анализ скриншота `photo_13_2026-02-20_09-52-54.jpg` (Управление питанием ErP)

- **ErP Ready [Выключено]:** Диретива ErP (Energy Related Products). Если включить (Enabled), компьютер будет потреблять менее 1 Вт в выключенном состоянии. Однако это отключит питание на USB-портах при выключении ПК (нельзя будет зарядить телефон), а также может отключить функцию Wake-on-LAN.
- **Восстановление после сбоя питания [Не включать сист...]:** Определяет поведение ПК после внезапного отключения и появления электричества. Может быть: "Power Off" (останется выключенным), "Power On" (включится сам), "Last State" (вернется в то состояние, в котором был до сбоя).

#### 13. Анализ скриншота `photo_14_2026-02-20_09-52-54.jpg` (Windows OS Configuration)

Настройки, специфичные для работы операционной системы Windows.

- **BIOS CSM/UEFI Mode [UEFI]:** Установлен чистый режим UEFI (без поддержки CSM).
- **GOP Information:** Информация о драйвере графического вывода GOP (Graphics Output Protocol), который необходим для работы UEFI.
- **Secure Boot:** Параметр безопасной загрузки. Должен быть включен для корректной работы Windows 11. На скриншоте, судя по пункту меню, он, вероятно, выключен (так как рядом нет галочки или статуса [Enabled]).
