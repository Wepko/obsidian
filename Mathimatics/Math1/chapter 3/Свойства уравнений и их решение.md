Отличный вопрос! Уравнения обладают целым набором свойств, которые позволяют их решать, то есть находить значения переменных (корни), которые обращают уравнение в верное равенство.

Эти свойства можно разделить на несколько групп.

### 1. Основные свойства равенств (для алгебраических уравнений)

Это самые важные свойства, на которых построено большинство методов решения уравнений. Они гласят, что если два выражения равны, то их можно:

*   **Свойство 1: Прибавление (или вычитание) одного и того же числа или выражения к обеим частям уравнения.**
    *   **Формулировка:** Если `a = b`, то `a + c = b + c`.
    *   **Пример:** `x - 5 = 10`. Прибавим 5 к обеим частям: `x - 5 + 5 = 10 + 5` → `x = 15`.

*   **Свойство 2: Умножение (или деление) обеих частей уравнения на одно и то же число (кроме нуля).**
    *   **Формулировка:** Если `a = b`, то `a * c = b * c` (и если `c ≠ 0`, то `a / c = b / c`).
    *   **Пример:** `3x = 12`. Разделим обе части на 3: `(3x)/3 = 12/3` → `x = 4`.
    *   **Важно:** Деление на ноль запрещено!

*   **Свойство 3: Возведение обеих частей уравнения в одну и ту же степень.**
    *   **Формулировка:** Если `a = b`, то `aⁿ = bⁿ` (для любого натурального n).
    *   **Пример:** `√x = 4`. Возведём обе части в квадрат: `(√x)² = 4²` → `x = 16`.
    *   **Внимание:** При возведении в чётную степень могут появиться посторонние корни (например, если было `x = 4`, то `x² = 16` имеет два корня: 4 и -4).

*   **Свойство 4: Если равны два выражения, то их можно подставлять друг в друга.**
    *   **Формулировка:** Если `a = b`, то `a` можно заменить на `b` в любом другом выражении (и наоборот).
    *   **Пример:** Дана система `y = 2x` и `x + y = 12`. Подставляем первое во второе: `x + 2x = 12` → `3x = 12` → `x = 4`.

### 2. Свойства, связанные с решением (корнями уравнения)

*   **Корни и множество решений:** Решение уравнения — это значение переменной (корень), которое обращает его в верное равенство. Множество всех корней называется **множеством решений**.
*   **Равносильность:** Два уравнения называются **равносильными**, если они имеют одно и то же множество решений. Преобразования из пункта 1 (кроме возведения в чётную степень и некоторых других случаев) обычно приводят к равносильному уравнению.
*   **Следствие:** Уравнение, которое получается в процессе преобразований исходного, но которое может иметь больше корней, называется **следствием**. Лишние корни называются **посторонними** и должны быть отброшены проверкой.
*   **Тождество:** Уравнение, которое верно при всех допустимых значениях переменной. Например, `(x + 1)² = x² + 2x + 1`.
*   **Противоречие (несовместное уравнение):** Уравнение, которое не имеет решений. Например, `x = x + 1`.

### 3. Свойства, связанные с типом уравнения

Разные типы уравнений имеют свои уникальные свойства.

#### Для квадратных уравнений (`ax² + bx + c = 0`):
*   **Дискриминант (`D = b² - 4ac`):** Определяет количество корней.
    *   `D > 0` — два различных корня.
    *   `D = 0` — один корень (два совпадающих).
    *   `D < 0` — нет действительных корней.
*   **Теорема Виета:** Для приведённого квадратного уравнения (`x² + px + q = 0`) сумма корней равна `-p`, а произведение корней равно `q`.

#### Для линейных уравнений (`ax + b = 0`):
*   Имеет ровно один корень, если `a ≠ 0` (`x = -b/a`).
*   Имеет бесконечно много решений (является тождеством), если `a = 0` и `b = 0`.
*   Не имеет решений (является противоречием), если `a = 0` и `b ≠ 0`.

### 4. Общие характеристики уравнения

*   **Степень уравнения:** Наивысшая степень переменной в уравнении.
    *   `2x + 5 = 0` — 1-я степень (линейное).
    *   `x² - 4 = 0` — 2-я степень (квадратное).
    *   Уравнение n-й степени имеет не более n корней (с учётом комплексных).
*   **Количество переменных:**
    *   `2x - 5 = 0` — уравнение с одной переменной.
    *   `x + y = 10` — уравнение с двумя переменными.
*   **Область допустимых значений (ОДЗ):** Множество всех значений переменной, при которых обе части уравнения имеют смысл. Например, в уравнении `1/(x-2) = 3` значение `x=2` не входит в ОДЗ, так как на ноль делить нельзя.

### Краткий итог:

1.  **Основные свойства равенств** — это "инструменты" для преобразования уравнений.
2.  **Свойства корней** описывают, сколько решений есть у уравнения и как они связаны между собой.
3.  **Специфические свойства** зависят от типа уравнения (линейное, квадратное, тригонометрическое и т.д.).
4.  **Общие характеристики** (степень, ОДЗ) помогают классифицировать уравнение и предсказать его поведение.
