Конечно, вот **итоговый перечень всех упомянутых инструментов и сервисов**, сгруппированный по категориям, с включением **Pencil.dev**.

---

### 🗺️ 1. Плагины внутри Figma
*Прямая конвертация дизайна в код из интерфейса Figma. Самый быстрый старт.*

| Инструмент | Особенности и целевые фреймворки |
| :--- | :--- |
| **Anima** | HTML, React, Vue, Storybook. Высокое качество кода, адаптивная верстка. |
| **Builder.io (Visual Copilot)** | React, Vue, Svelte, Flutter, HTML. AI-промпты, Tailwind CSS. |
| **Locofy** | React, React Native, HTML/CSS, Flutter. Ориентир на production-ready код. |
| **TeleportHQ** | Генерация кода + визуальный редактор для совместной доработки. |
| **DhiWise** | React, Next.js, Flutter. Генерация модульного кода для приложений. |
| **Figment** | React, Vue, HTML. Работает как мост между Figma и IDE. |
| **Kombai** | AI-агент с глубокой интеграцией в существующий проект и кодовую базу. |
| **CodeParrot** | «Понимает» ваш код, переиспользует компоненты и соблюдает стиль проекта. |

### ⚙️ 2. AI-агенты и MCP-серверы (Связка Figma + IDE)
*Интеграция прямо в среду разработки. AI «видит» макет и генерирует код в вашем проекте.*

| Инструмент | Тип и назначение |
| :--- | :--- |
| **Pencil.dev** | 🚀 **Новый гибрид**: MCP-сервер + векторный холст в IDE. Стирает грань между дизайном и кодом. Работает в VS Code / Cursor. |
| **Официальный Figma MCP Server** | Базовый MCP-сервер от Figma для доступа к слоям и стилям через AI (Claude, Cursor). |
| **f2c-mcp-server** | Специализированный сервер для конвертации в React с Tailwind CSS. |
| **figma-flutter-mcp** | MCP-сервер для генерации кода под Flutter. |
| **Flowbite MCP Server** | Сервер для Tailwind CSS и компонентов Flowbite. |
| **Kailash G. MCP** | Сервер для React Native. |
| **Cursor / Windsurf / Claude Desktop / Claude Code** | IDE и AI-ассистенты, которые из коробки умеют подключаться к Figma MCP-серверам. |

### 🌐 3. Онлайн-сервисы и No-Code/Low-Code платформы
*Загрузка ссылки или скриншота на веб-платформу. Часто с возможностью дальнейшего визуального редактирования.*

| Инструмент | Особенности |
| :--- | :--- |
| **Figma Make** | Встроенный в Figma конструктор приложений по макету или тексту. |
| **V0 by Vercel** | Генерация UI-компонентов для React/Next.js + Tailwind по изображению или промпту. |
| **Lovable** | Создание полноценных веб-приложений с бэкендом (Supabase) по дизайну. |
| **App2.dev** | Конвертация Figma-макетов в мобильные приложения на React Native/Expo. |
| **Relume** | Библиотека готовых блоков для Figma и Webflow с AI-подбором. |
