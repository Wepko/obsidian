Вот это интересно! Проблема ясна. У вас настроено **только одно направление** переключения (`switch-input-source`), но нет обратного переключения.

## Решение: Добавьте оба направления

```bash
# 1. Удалите текущую настройку (опционально)
gsettings reset org.gnome.desktop.wm.keybindings switch-input-source
gsettings reset org.gnome.desktop.wm.keybindings switch-input-source-backward

# 2. Настройте ОБЕ комбинации на Alt+Shift
gsettings set org.gnome.desktop.wm.keybindings switch-input-source "['<Alt>Shift_L', '<Alt>Shift_R', '<Shift>Alt_L', '<Shift>Alt_R']"
gsettings set org.gnome.desktop.wm.keybindings switch-input-source-backward "['<Shift>Alt_L', '<Shift>Alt_R', '<Alt>Shift_L', '<Alt>Shift_R']"
```
