В Python консоль REPL обычно называется:

1. **Интерактивная оболочка Python** (Python Interactive Shell)
2. **Интерпретатор Python в интерактивном режиме**
3. **Python REPL** (Read-Eval-Print Loop)

Как запустить:
```bash
python  # или python3
```

Или для более продвинутой версии:
```bash
python -i  # запуск скрипта и переход в интерактивный режим
```

Внутри кода можно также запустить REPL с помощью:
```python
import code
code.interact(local=locals())
```

**Другие варианты REPL для Python:**
- **IPython** - более продвинутая интерактивная оболочка
- **IDLE** - встроенная среда разработки Python с REPL
- **Jupyter Notebook/IPython Notebook** - интерактивные блокноты
- **bpython** - альтернативная интерактивная оболочка
- **ptpython** - еще один улучшенный REPL

Обычно, когда говорят "зайди в Python консоль", имеют в виду стандартную интерактивную оболочку, которая открывается при запуске команды `python` без аргументов.
