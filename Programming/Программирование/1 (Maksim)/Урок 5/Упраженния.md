#### **2. –°—Ç–æ–∏–º–æ—Å—Ç—å –∑–∞–∫—É–ø–∫–∏ 60 –∫–Ω–∏–≥ —Å–æ —Å–∫–∏–¥–∫–æ–π –∏ –¥–æ—Å—Ç–∞–≤–∫–æ–π**  
- –¶–µ–Ω–∞ –∫–Ω–∏–≥–∏: `249.50` —Ä—É–±.  
- –°–∫–∏–¥–∫–∞: `40%` ‚Üí —Ü–µ–Ω–∞ —Å–æ —Å–∫–∏–¥–∫–æ–π: `249.50 * 0.6 = 149.70` —Ä—É–±.  
- –î–æ—Å—Ç–∞–≤–∫–∞:  
  - –ü–µ—Ä–≤–∞—è –∫–Ω–∏–≥–∞: `100` —Ä—É–±.  
  - –û—Å—Ç–∞–ª—å–Ω—ã–µ `59` –∫–Ω–∏–≥: `49.50` —Ä—É–±. –∑–∞ –∫–∞–∂–¥—É—é.  

```python

# –¶–µ–Ω–∞ —Å–æ —Å–∫–∏–¥–∫–æ–π

# –°—Ç–æ–∏–º–æ—Å—Ç—å –∫–Ω–∏–≥

# –î–æ—Å—Ç–∞–≤–∫–∞

# –ò—Ç–æ–≥

print(total_cost)  # 11987.5
```
**–û—Ç–≤–µ—Ç:** `11987.5` —Ä—É–±.

#### **3. –í—Ä–µ–º—è –≤–æ–∑–≤—Ä–∞—â–µ–Ω–∏—è –¥–æ–º–æ–π –ø–æ—Å–ª–µ –ø—Ä–æ–±–µ–∂–∫–∏**

  1. **1 –∫–º** ‚Äî –ª–µ–≥–∫–∏–π —Ç–µ–º–ø (`8 –º–∏–Ω. 15 —Å–µ–∫.` = `495 —Å–µ–∫.`).  
  2. **3 –∫–º** ‚Äî —Å—Ä–µ–¥–Ω–∏–π —Ç–µ–º–ø (`7 –º–∏–Ω. 12 —Å–µ–∫.` = `432 —Å–µ–∫.` –∑–∞ –∫–º).  
  3. **1 –∫–º** ‚Äî –ª–µ–≥–∫–∏–π —Ç–µ–º–ø (`495 —Å–µ–∫.`).  

