## **Задачи: Инструкция (statement) или Выражение (expression)?**

Для каждой конструкции определите:
1. Это инструкция (S), выражение (E), или выражение-инструкция (E-S)?
2. Если выражение — можно ли его использовать справа от `=`?
3. Если инструкция — можно ли её использовать как аргумент функции?

## **Ответы и подробные объяснения:**

### **Часть 1:**

```python
# 1. x = 10
# Ответ: ИНСТРУКЦИЯ (S)
# Это инструкция присваивания. Нельзя: print(x = 10) ❌

# 2. x + 5
# Ответ: ВЫРАЖЕНИЕ как инструкция (E-S)
# Выражение, которое Python обрабатывает как инструкцию.
# Можно: y = x + 5 ✅

# 3. print("Hello")
# Ответ: ВЫРАЖЕНИЕ как инструкция (E-S)
# Вызов функции — выражение, но часто используется как инструкция.
# Можно: y = print("Hello") (y будет None)

# 4. if x > 0: pass
# Ответ: ИНСТРУКЦИЯ (S)
# Инструкция условия. Нельзя: y = if x > 0: pass ❌

# 5. "Hello, World!"
# Ответ: ВЫРАЖЕНИЕ как инструкция (E-S)
# Строковый литерал — выражение. Бесполезно как инструкция.
# Можно: y = "Hello, World!" ✅

# 6. len("hello")
# Ответ: ВЫРАЖЕНИЕ как инструкция (E-S)
# Вызов функции — выражение.
# Можно: y = len("hello") ✅

# 7. import math
# Ответ: ИНСТРУКЦИЯ (S)
# Инструкция импорта. Нельзя: y = import math ❌

# 8. def greet(): return "Hi"
# Ответ: ИНСТРУКЦИЯ (S)
# Определение функции — инструкция. Нельзя: y = def greet(): ... ❌

# 9. lambda x: x * 2
# Ответ: ВЫРАЖЕНИЕ (E)
# Лямбда создает анонимную функцию (объект функции).
# Можно: y = lambda x: x * 2 ✅

# 10. for i in range(5): print(i)
# Ответ: ИНСТРУКЦИЯ (S)
# Цикл for — инструкция. Нельзя: y = for i in range(5): print(i) ❌
```

### **Задание 3: Найдите все инструкции в этом коде**
```python
def calculate(x, y):
    if x > 0:
        result = x * y
    else:
        result = 0
    return result

total = calculate(10, 5)
print(f"Result: {total}")
```

**Ответ:** 8 инструкций:
1. `def calculate(x, y):` (инструкция определения функции)
2. `if x > 0:` (инструкция if)
3. `result = x * y` (инструкция присваивания, внутри if)
4. `else:` (часть инструкции if)
5. `result = 0` (инструкция присваивания, внутри else)
6. `return result` (инструкция return)
7. `total = calculate(10, 5)` (инструкция присваивания)
8. `print(f"Result: {total}")` (выражение как инструкция)

## **Ключевые правила для самопроверки:**

1. **Можно ли это использовать справа от `=`?** 
   - Да → выражение (или содержит выражение)
   - Нет → инструкция

2. **Начинается ли с ключевого слова, за которым идет двоеточие (`:`)?**
   - Да → скорее всего инструкция

3. **Возвращает ли это значение, которое можно использовать дальше?**
   - Да → выражение
   - Нет → инструкция (или выражение, возвращающее None)

4. **Можно ли это написать как аргумент функции без изменения синтаксиса?**
   - Да → выражение
   - Нет → инструкция
