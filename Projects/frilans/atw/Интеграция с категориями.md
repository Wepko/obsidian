## 6. Обновляем существующие товары, привязанные к старым категориям

```sql
-- Товары из категории "Шины" (62) остаются - они теперь в структуре "Шины и диски/Шины"
-- Товары из категории "Диски" (63) остаются - они теперь в структуре "Шины и диски/Диски"

-- Товары из "Масла моторные" (65) переносим в новую категорию (205)
UPDATE shop_catalogue_products SET category_id = 205 WHERE category_id = 65;

-- Товары из "Масла трансмиссионные и ГУР" (66) - разделяем:
-- Сначала определяем какие товары относятся к трансмиссионным, а какие к ГУР
-- В примере товар с id=99 - трансмиссионное масло ENEOS -> в категорию 207
UPDATE shop_catalogue_products SET category_id = 207 WHERE id = 99;
-- Остальные товары с category_id = 66 будут перенесены как масло для ГУР (206)
UPDATE shop_catalogue_products SET category_id = 206 WHERE category_id = 66 AND id != 99;
```

## 7. Опционально: обновляем счетчики (count) для всех категорий

```sql
-- Обновляем счетчики для всех категорий (рекурсивно)
UPDATE shop_catalogue_categories c
SET c.count = (
    SELECT COUNT(*) FROM shop_catalogue_products p 
    WHERE p.category_id = c.id
)
WHERE c.level = 3;

UPDATE shop_catalogue_categories c
SET c.count = (
    SELECT SUM(count) FROM shop_catalogue_categories child 
    WHERE child.parent = c.id
)
WHERE level IN (1, 2);
```
