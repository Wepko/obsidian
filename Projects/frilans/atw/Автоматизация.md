### Шаг 2: Создаём скрипт деплоя

```bash
cd ~/prj/atw/docpart/atw.parts
nano deploy.sh
```

**Скопируйте этот полный скрипт:**

```bash
#!/bin/bash

# Цвета для вывода
RED='\033[0;31m'
GREEN='\033[0;32m'
YELLOW='\033[1;33m'
BLUE='\033[0;34m'
NC='\033[0m' # No Color

# Настройки
SERVER_USER="aabarap7"
SERVER_HOST="forest"
SERVER_PATH="~/atw.parts/public_html"
LOCAL_PATH="src/"

echo -e "${BLUE}════════════════════════════════════════${NC}"
echo -e "${GREEN}🚀 Деплой проекта на сервер${NC}"
echo -e "${BLUE}════════════════════════════════════════${NC}"

# 1. Проверяем, что мы в правильной папке
echo -e "\n${YELLOW}[1/5] Проверка...${NC}"
if [ ! -d "$LOCAL_PATH" ]; then
    echo -e "${RED}❌ Ошибка: папка $LOCAL_PATH не найдена${NC}"
    echo "Запустите скрипт из корня проекта"
    exit 1
fi
echo -e "${GREEN}✓ Папка проекта найдена${NC}"

# 2. Сохраняем изменения в Git
echo -e "\n${YELLOW}[2/5] Сохранение в Git...${NC}"
git add .
if git diff --cached --quiet && git diff --quiet; then
    echo -e "${GREEN}✓ Нет изменений для сохранения${NC}"
else
    git commit -m "Auto-commit before deploy $(date +%Y-%m-%d_%H-%M-%S)"
    echo -e "${GREEN}✓ Изменения сохранены${NC}"
fi

# 3. Создаём бэкап на сервере
echo -e "\n${YELLOW}[3/5] Создание бэкапа на сервере...${NC}"
BACKUP_NAME="backup_$(date +%Y%m%d_%H%M%S).tar.gz"
if ssh ${SERVER_USER}@${SERVER_HOST} "cd ~/atw.parts && tar -czf ${BACKUP_NAME} public_html/ 2>/dev/null"; then
    echo -e "${GREEN}✓ Бэкап создан: ${BACKUP_NAME}${NC}"
else
    echo -e "${RED}❌ Ошибка создания бэкапа${NC}"
    exit 1
fi

# 4. Показываем, что будет удалено (dry run)
echo -e "\n${YELLOW}[4/5] Проверка изменений...${NC}"
echo "Будут удалены на сервере:"
rsync -avz --delete --dry-run \
    --exclude='config.php' \
    --exclude='.env' \
    --exclude='*.log' \
    --exclude='backup_*.tar.gz' \
    ${LOCAL_PATH} \
    ${SERVER_USER}@${SERVER_HOST}:${SERVER_PATH}/ 2>/dev/null | grep "deleting" | head -10

if [ $? -eq 0 ]; then
    echo ""
    echo -e "${YELLOW}Продолжить деплой? (y/n)${NC}"
    read -r CONFIRM
    if [ "$CONFIRM" != "y" ]; then
        echo -e "${RED}❌ Деплой отменён${NC}"
        exit 1
    fi
fi

# 5. Синхронизация файлов
echo -e "\n${YELLOW}[5/5] Синхронизация файлов...${NC}"
rsync -avz --delete \
    --exclude='config.php' \
    --exclude='.env' \
    --exclude='*.log' \
    --exclude='backup_*.tar.gz' \
    -e "ssh" \
    ${LOCAL_PATH} \
    ${SERVER_USER}@${SERVER_HOST}:${SERVER_PATH}/

if [ $? -eq 0 ]; then
    echo -e "${GREEN}✓ Файлы скопированы${NC}"
else
    echo -e "${RED}❌ Ошибка копирования${NC}"
    exit 1
fi

# Очистка старых бэкапов (оставляем 5 последних)
echo -e "\n${YELLOW}Очистка старых бэкапов...${NC}"
ssh ${SERVER_USER}@${SERVER_HOST} "cd ~/atw.parts && ls -t backup_*.tar.gz 2>/dev/null | tail -n +6 | xargs rm -f 2>/dev/null || true"

# Проверка сайта
echo -e "\n${YELLOW}Проверка сайта...${NC}"
HTTP_CODE=$(curl -s -o /dev/null -w "%{http_code}" https://atw.parts 2>/dev/null || echo "000")
if [ "$HTTP_CODE" = "200" ] || [ "$HTTP_CODE" = "301" ] || [ "$HTTP_CODE" = "302" ]; then
    echo -e "${GREEN}✓ Сайт отвечает (HTTP $HTTP_CODE)${NC}"
else
    echo -e "${YELLOW}⚠ Внимание! Сайт вернул код $HTTP_CODE${NC}"
fi

echo -e "\n${BLUE}════════════════════════════════════════${NC}"
echo -e "${GREEN}✅ Деплой завершён!${NC}"
echo -e "${BLUE}════════════════════════════════════════${NC}"

# Показываем последние бэкапы
echo -e "\n${YELLOW}Последние бэкапы на сервере:${NC}"
ssh ${SERVER_USER}@${SERVER_HOST} "ls -lh ~/atw.parts/backup_*.tar.gz 2>/dev/null | tail -3"
```

### Шаг 5: Упрощённая версия скрипта (без подтверждений)

Если хотите максимально быстрый деплой без лишних вопросов:

```bash

```

```bash
#!/bin/bash

# Быстрый деплой без подтверждений
SERVER="aabarap7@forest"
PATH_SERVER="~/atw.parts/public_html"

echo "🚀 Быстрый деплой..."

# Сохраняем в Git
git add .
git commit -m "Quick deploy $(date +%H:%M:%S)" 2>/dev/null || true

# Бэкап
ssh $SERVER "cd ~/atw.parts && tar -czf backup_$(date +%Y%m%d_%H%M%S).tar.gz public_html/"

# Синхронизация с удалением
rsync -avz --delete \
    --exclude='config.php' \
    --exclude='.env' \
    src/ \
    $SERVER:$PATH_SERVER/

echo "✅ Готово!"
```

```bash

```

## Как пользоваться

### Обычный деплой (с проверками):
```bash
./deploy.sh
```

### Быстрый деплой (без вопросов):
```bash
./deploy-quick.sh
```

### Только посмотреть, что изменится (без реального деплоя):
```bash
rsync -avz --delete --dry-run src/ aabarap7@forest:~/atw.parts/public_html/
```

## Что делает скрипт:

1. **Проверяет** - что мы в правильной папке
2. **Сохраняет** - все изменения в Git
3. **Бэкапит** - текущую версию на сервере
4. **Показывает** - какие файлы будут удалены
5. **Спрашивает** - подтверждение (только в полной версии)
6. **Синхронизирует** - копирует новые файлы и удаляет лишние
7. **Чистит** - старые бэкапы (оставляет 5 последних)
8. **Проверяет** - что сайт работает
