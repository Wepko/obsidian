Структура URL:

- masla-i-teh-zhidkosti — alias категории верхнего уровня
- masla — alias категории второго уровня
- motornoe-maslo — alias категории третьего уровня
- castrol-edge-5w-30-1l — alias товара

Как это работает:
1. Маршрутизация — запросы проходят через .htaccess → index.php → core/dp_core.php
2. Роутинг — в config.php строка 51: public $product_url = 'alias' указывает, что URL товара строится из alias, а не из ID
3. Данные в БД — сами категории и товары (включая alias castrol-edge-5w-30-1l) хранятся в MySQL базе данных, а не в исходниках
4. Генерация ссылок — в content/shop/catalogue/generate_products_objects_by_sql.php строки 78-84:
    - Если product_url == "id" → URL: /category_url/product_id
    - Иначе → URL: /category_url/product_alias
5. Вывод товара — content/shop/catalogue/product_page_for_customer.php подключает printProduct_Info.php

Ключевые файлы для маршрутизации:
- core/dp_core.php — основной роутинг
- content/shop/catalogue/get_catalogue_tree.php — дерево категорий с alias
- content/shop/catalogue/generate_products_objects_by_sql.php — генерация URL для товаров

Конкретный товар castrol-edge-5w-30-1l — это alias товара Castrol Edge 5W-30 1L, который хранится в таблице shop_storages_data или связанных таблицах продуктов в базе данных MySQL.
